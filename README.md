
Home page for Respo project
----

http://respo.site

### Develop

Based on CoffeeScript, React, Gulp.

Install dependency:

```bash
npm i
```

Run in development mode:

```bash
# setup static site for `build/` with a domain
gulp dev
```

```bash
gulp build-all
# all files in `build/`
```
